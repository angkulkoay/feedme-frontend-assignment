# Order Controller Dashboard

Frontend prototype for a cooking bot order controller with VIP queue priority, multi-bot processing, and live order state transitions.

## Live Demo

- Production: `https://feedme-frontend-assignment.vercel.app`

## Stack

- Next.js
- TypeScript
- Tailwind CSS
- React hooks

## Local Development

```bash
npm install
npm run dev
```

## Verification

```bash
npm run lint
npm run build
```

## Project Notes

- All state is stored in React memory.
- No backend, database, or authentication is used.
- Automatic deployments run from GitHub Actions to Vercel on pushes to `main`.
